# fast-nav · browser demo

**Live demo: https://fastnav.jacobvm.com/**

Interactive point-navigation demo: pick a held-out indoor scene, click anywhere to set a
goal, and watch a trained policy navigate there in real time. Everything — lidar
simulation, physics, and GRU policy inference — runs in plain JavaScript in your browser.
No server, no WebGL, no dependencies.

The policy is a GRU-256 PPO agent (~98% success on held-out scenes) trained in
**fast-nav**, an MLX/Metal lidar navigation sim for Apple silicon that steps tens of
millions of environments per second. This repo holds only the exported static demo; the
training code isn't public yet. The JS sim is verified to match the MLX training sim
exactly (`test/parity.mjs` replays an MLX-generated trajectory: zero position divergence
over a full episode).

## Controls

- **click** — set a navigation goal
- **shift-click** — teleport the robot
- **policy** — `PPO · clean-trained` vs `PPO · noise-trained (DR 1.0)`
- **noise** — sim2real noise stack (odometry drift, heading error, lidar noise/dropouts,
  actuation noise) at off / ½× / 1× / 2× realistic levels. With noise on, the hollow
  dashed ring shows where the robot *believes* it is from odometry — the policy only
  sees that belief.
- **speed / pause** — playback control (physics always steps at the training dt = 0.1 s)

Scenes are held-out [ReplicaCAD](https://aihabitat.org/datasets/replica_cad/) apartments
and [ProcTHOR](https://procthor.allenai.org/) houses the policy never saw in training.

## Run locally

```bash
python3 -m http.server          # then open http://localhost:8000
node test/parity.mjs            # JS vs MLX sim/policy parity check
```

## Layout

- `index.html`, `js/` — the whole app (canvas renderer, sim port, policy inference)
- `policies.json`, `policies/*.bin` — policy weights as raw float32 + tensor manifest
- `scene_index.json`, `scenes/*.png` — 1-bit occupancy grids; the browser recomputes
  the signed Euclidean distance field on load
- `test/` — parity fixture generated by the MLX training sim
